🚀 النشر على Render
══════════════════════════════════════════════════════

☐ إنشاء حساب:
  ✓ اذهب إلى https://render.com
  ✓ سجل حساب جديد (استخدم GitHub)

☐ إنشاء Web Service:
  ✓ اضغط "New +" → "Web Service"
  ✓ اربط حساب GitHub
  ✓ اختر المستودع
  ✓ Render سيكتشف render.yaml تلقائياً

☐ إضافة متغيرات البيئة:
  ✓ اضغط "Environment"
  ✓ أضف BOT_TOKEN
  ✓ أضف FIREBASE_CREDENTIALS (JSON كامل في سطر واحد)
  ✓ أضف SECRET_KEY
  ✓ أضف SITE_URL (https://your-app.onrender.com)

☐ النشر:
  ✓ اضغط "Create Web Service"
  ✓ انتظر اكتمال البناء (2-5 دقائق)
  ✓ انسخ رابط التطبيق

☐ تحديث SITE_URL:
  ✓ عد إلى Environment في Render
  ✓ حدّث قيمة SITE_URL برابط التطبيق الفعلي
  ✓ احفظ التغييرات
